# C Mini Calculator

A simple console-based calculator written in C.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Uses switch-case logic

## How to Run
1. Compile the program using a C compiler:
   gcc calculator.c -o calculator
2. Run the executable:
   ./calculator

## Note
- Designed for Windows console
- Uses `system("color 4")`
